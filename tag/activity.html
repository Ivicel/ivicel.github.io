Android Activity生命周期和启动模式

Activity 的生命周期

一般 Activity 生命周期为 onCreate() -> onStart() -> onResume() -> onPause() -> onStop() -> onDestroy()

官方的生命周期图:

需要注意的几个点:

  1. 当一个 Activity A 启动加一个 Activity B 时, 只有 A 的 onPause 执行完返回时, B 才开始执行onCreate onStart, 直到 onResume显示在前台, A 会执行 onStop onDestroy, 所以在 onPause不能执行太耗时的操作, 以便让另一个 Activity 快点显示到前台
  2. Activity 会在应用发生变更时(常见的屏幕旋转, 输入框显示/隐藏, 屏幕大小变化 …
